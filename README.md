# 🧮 Flutter Calculator App

A modern **Flutter-based commercial calculator application** designed for daily business and utility needs.
This app goes beyond basic calculations by including **GST calculation, profit analysis, temperature conversion, and shopkeeper notes**, making it a practical tool for real-world usage.

---

## 🚀 Project Overview

This project is a multi-functional calculator built using **Flutter (Dart)** that combines essential business tools into a single application.

It is designed with a focus on:

* 📱 Clean UI/UX
* ⚡ Fast performance
* 🧠 Practical real-world features

---

## ✨ Key Features

### 🧮 Basic Calculator

* Perform standard arithmetic operations
* Responsive and user-friendly interface

### 💰 GST Calculator

* Calculate GST instantly
* Add/remove GST with accurate results
* Useful for shopkeepers and businesses

### 📈 Profit Calculator

* Calculate profit and loss easily
* Helps in quick business decision making

### 🌡 Temperature Converter

* Convert between different temperature units

  * Celsius
  * Fahrenheit
  * Kelvin

### 📝 Shopkeeper Notes

* Save important notes inside the app
* Persistent storage using SharedPreferences
* Helps track small business records

---

## 🏗️ Tech Stack

* **Flutter** – Cross-platform UI framework
* **Dart** – Programming language
* **Provider** – State management
* **SharedPreferences** – Local data storage

Flutter enables building **high-performance apps for Android & iOS from a single codebase** ([flutter.fizikamind.com][1])

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Sauravx4/flutter-calculator-project.git
cd flutter-calculator-project
```

### 2️⃣ Install dependencies

```bash
flutter pub get
```

### 3️⃣ Run the app

```bash
flutter run
```
